如何判断论文水平

认可度与出版渠道

  • 认可度较高的期刊：ACM Transaction on … 、IEEE Transaction on …

  • 出版商 / 学会：ACM、IEEE、Elsevier、AAAI、CCF（中国计算机学会）

论文类型与分类

  • 论文类型：期刊论文、会议论文

  • 中科院分区：一区、二区、三区、四区

  • CCF 分类：A 类、B 类、C 类

检索与索引

SCI、EI

论文级别参考标准

中国计算机学会（CCF）推荐目录

  • A 类 > B 类 > C 类 > 其他

    • https://www.ccf.org.cn/Academic_Evaluation/By_category/

中科院分区

  • 1 区 > 2 区 > 3 区 > 4 区 > 其他

    • http://www.letpub.com.cn/index.php?page=journalapp&view=search
    • https://www.fenqubiao.com/
    • 中科院分区查询公众号

如何阅读论文

阅读原则

大量阅读论文，夯实科研基础

建议：每周精读 1 篇，粗读 ≥5 篇

阅读类型

预热性阅读

知识输入、了解国内外研究现状、跟进领域最新进展

批判性阅读

判断研究是否合理、分析论证是否充分

创造性阅读

获取灵感、设计新算法、发现并解决新问题

精读重点

文章解决的初始问题、创新点与核心思想、新模型 / 新算法设计思路、实验设计与验证方法、结果分析与优缺点评价

泛读目标

把握研究方向整体情况、快速筛选高质量论文、阅读题目摘要、浏览章节结构与模型框架

写作顺序和思路

论文基本模块

  • Abstract
  • Introduction
  • Related Work
  • Main Results（Theory / Model / Algorithm）
  • Experiments
  • Discussion
  • Conclusion

摘要与引言写作思路

  1. 介绍研究背景
  2. 指出现有场景中的问题
  3. 回顾已有解决方案
  4. 分析现有方案不足
  5. 提出本文解决方案
  6. 进行性能与安全性分析
  7. 总结方案优势

论文写作逻辑

3W2H 写作法

1.Why：为什么做？

Motivation、Novelty、Significance

2.What：做了什么？

Contribution

3.How：怎么做？

Theorem / Model / Algorithm、清晰性、正确性、可复现性

4.How Much：效果如何？

Experiments、Discussion

5.What Then：接下来做什么？

Conclusion、启示与未来工作

Motivation 展开方式

理论或应用需求、以往工作的不足、解决问题的潜在影响

工具与实现

写作：TeXstudio + TeXlive

文献管理：DBLP

语言润色：ChatGPT

代码复现：GitHub + 自编代码

实验环境：Python

作图：Flaticon + PPT + Origin

投稿方式：学校邮箱 + 投稿系统

如何面对科研焦虑

常见心理与应对方式

想不出来 → 那就安心地做

紧张 → 那就紧张地做

没有动力 → 那就没有动力地做

觉得自己菜 → 那就继续学着做

觉得没准备好 → 那就准备着做

博士修行的“冰山模型”

看得见的：论文、成果、奖励

看不见的：阅读、思考、失败、修改、坚持
